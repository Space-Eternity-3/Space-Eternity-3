New features:

- Added shield potion, obtainable and possible to produce on copper facilities.
  - Shield potion removes your effect and gives you a shield, which makes you indestructible and unable to shoot for 2 seconds.

Old feature changes:



Artistic changes:

- Geyzer volume was reduced: 0.25 -> 0.16

UI changes:

- Text "Version: " was removed from the universe selection menu. The version name will be displayed alone from now.
- Text "Recharge artefact using star" changed to "Recharge artefact near star".
- Music and Volume settings can now be set to all whole values from 0% to 100%.

Configurables:

- Added gameplay variables in datapack format:
  - shield_potion_duration: 2;

Default datapack changes:

- To fit new datapack format:
  - Added new gameplay variables.

Technical changes:

- Anti-aliasing increased from x4 to x8 on ultra (default) settings.
- Reworked the impulse shield system to fit the potion shield too.

Bug fixes:

- Chat was positioned wrongly, when user had other than 16:9 screen ratio.
- Impulse "god-mode" is no longer possible through cheats.
- SND_fire sound (for fire and wind bullets) had some weird artefacts at the end
